## Como rodar

Clone o projeto

```bash
  git clone https://github.com/Thierryvil/react-alurakut
```

Entre no diretorio

```bash
  cd react-alurakut
```

Instale as dependecias

```bash
  npm install 
  yarn install
```

Inicie o servidor

```bash
  yarn dev
```
